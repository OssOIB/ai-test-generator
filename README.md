```
└── .env.example                 # API key template
```

### How It Works

| Step | What Happens |
|---|---|
| **1. Crawl** | Launches a headless Chromium browser, navigates to the URL, extracts forms, buttons, links, headings, ARIA labels, and `data-testid` attributes |
| **2. Analyze** | Structures the page data into a detailed prompt describing every interactive element |
| **3. Generate** | Streams the prompt to Claude Sonnet 4.6 — the static system prompt is **prompt-cached** for ~10× cheaper repeat calls |
| **4. Run** | Saves the generated TypeScript test, executes it with `npx playwright test`, and parses pass/fail counts |

### AI Integration Details

- **Model:** `claude-sonnet-4-6` (replaces deprecated `claude-sonnet-4-20250514`)
- **Prompt caching:** System prompt marked `cache_control: ephemeral` — 5-minute TTL, reads at 0.1× input cost
- **Streaming:** Test code streams to terminal in real-time via SSE
- **Smart extraction:** Handles markdown-fenced and raw TypeScript responses

## Quick Start

### 1. Install

```bash
# Run directly via npx (no install needed)
npx ai-test-generator https://example.com

# Or install globally
npm install -g ai-test-generator
ai-test-generator https://example.com
```

### 2. Configure API Key

```bash
cp .env.example .env
# Edit .env and set your ANTHROPIC_API_KEY
```

Get your key at [console.anthropic.com](https://console.anthropic.com).

### 3. Install Playwright Browsers

```bash
npx playwright install chromium
```

### 4. Generate Tests

```bash
# Full pipeline: crawl → generate → run
ai-test-generator https://www.saucedemo.com

# Generate only, save to specific file
ai-test-generator https://example.com -o tests/example.spec.ts --no-run

# Use Firefox for crawling
ai-test-generator https://example.com --browser firefox

# Save generated test to a named file and run it
ai-test-generator https://myapp.com -o tests/myapp.spec.ts
```

## CLI Reference

```
Usage: ai-test-generator <url> [options]

Arguments:
  url                    URL to analyze and generate Playwright tests for

Options:
  -o, --output <path>    Save generated test to a specific file
  --no-run               Generate tests without running them
  -b, --browser <b>      Browser for crawling: chromium | firefox | webkit (default: chromium)
  -V, --version          Output the version number
  -h, --help             Display help
```

## Environment Variables

| Variable | Default | Description |
|---|---|---|
| `ANTHROPIC_API_KEY` | — | **Required.** Your Anthropic API key |
| `AI_MODEL` | `claude-sonnet-4-6` | Claude model to use |
| `CRAWLER_BROWSER` | `chromium` | Default browser for page crawling |
| `MAX_TOKENS` | `8000` | Max tokens for test generation |

## CI/CD

GitHub Actions runs on every push:

1. **Build & Type Check** — compiles TypeScript, verifies types
2. **E2E Generate & Run** — (requires `ANTHROPIC_API_KEY` secret):
   - Generates tests for `https://www.saucedemo.com`
   - Runs them with Playwright
   - Uploads HTML report as artifact

Set the secret in GitHub: **Settings → Secrets → Actions → `ANTHROPIC_API_KEY`**.
